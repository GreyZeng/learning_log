# Web 应用程序 基于: Django

[Python编程：从入门到实践（第3版）](https://www.ituring.com.cn/book/3038)示例项目

首先需要建立数据库

```pip
python -m venv ll_env
```

激活数据库， Windows 下

```cmd
ll_env\Scripts\activate
```

Mac 下

```sh
source ll_env/bin/activate
```

安装Django

```pip
pip install --upgrade pip
pip install django
```

使用bootstrap美化

```pip
.\ll_env\Scripts\pip.exe install django-bootstrap5
```
